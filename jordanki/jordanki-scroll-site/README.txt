JORDANKI PROFESSIONAL STOREFRONT

Open index.html using VS Code Live Server or any local web server.

Included:
- Full-screen responsive scroll-controlled hero video
- Animated water/ripple canvas layer
- Men, women, new arrival and best-seller filters
- Product quick view and 3D-style drag interaction
- Working cart, totals, demo checkout and newsletter
- Responsive mobile/tablet/desktop design

Product imagery:
- User-provided/local images are bundled in assets.
- Additional catalogue imagery is loaded from Pexels image CDN, with a unique local fallback for every card.
- Replace catalogue image URLs with your licensed supplier images before commercial launch.

Payment:
Checkout is a functional front-end demo. Connect Shopify, WooCommerce, Stripe or PayFast credentials for real transactions.
